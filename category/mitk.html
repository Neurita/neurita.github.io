How to compile MITK on Ubuntu

2015/09/01

Prerequisites

  • GIT
  • CMake (version 3.2 or higher. Current stable 3.3)
  • Qt 5.x
  • Make sure that you select a Qt version which provides the right OpenGL-enabled packages for your architecture and compiler

Building
